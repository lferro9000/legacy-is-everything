What is legacy?

  • noun
  • an amount of money or property left to someone in a will.

  • adjective
  • denoting or relating to software or hardware that has been superseded but is difficult to replace because of its wide use.

Legacy code

  • legacy code is code without tests (Michael Feathers)

    • Michael Feathers' Working Effectively with Legacy Code (ISBN 0-13-117705-2)

    Everything is

  • code that doesn't change is legacy

    • You have it!

    • code inherited from others
    • code from dependencies

    It's everywhere!

    • even in code just written!

    Legacy as a value

    • The code exists in production
    • Generates revenue for the company

    Legacy as a constraint

    • code is hard to change
    • takes time to add features
    • even harder to test and validate
    • increased costs at every interaction
